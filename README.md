# Shizuku Engine 4.0 α — v0.6 “Paint Clump”

**Shizuku Engine** は、子どもが「本物の絵の具を指で触っている感覚」をデジタル上で再現することを目指す、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## v0.6 Paint Clump 1.0

- 粒子間に壊れる内部結合（clump bonds）を追加
- 弱い力では塊を維持し、強い引っ張りと高い相対速度が重なった時だけ結合が切れる
- 結合の少ない塊の端ほどちぎれやすくし、飛沫が塊の縁から発生する構造へ変更
- ゆっくりこねた時は、近接した粒子同士が少しずつ再結合する
- 各粒子に `clumpId` と `pigment` を保持し、将来のマーブル混色へつなげる
- v0.3の性能制限、v0.4のGrip、v0.5のViscosityを維持

## テスト方法

1. `index.html` を開く
2. 中央の塊をゆっくり円を描くようにこね、まとまりが維持されるか確認
3. 塊の端をゆっくり引き、首状に伸びるか確認
4. 同じ場所を素早く強く引き、端だけが小さくちぎれるか確認
5. ちぎれた部分を本体へ押し戻し、ゆっくりこねると再びまとまりやすくなるか確認
6. 「えのぐをたす」を上限まで押しても停止しないことを確認

## 合否の基準

- 通常の操作では飛沫が増えすぎない
- 弱い操作では塊の形が保たれる
- 強い操作でのみ端がちぎれる
- 長時間触っても著しい処理落ちや停止がない
