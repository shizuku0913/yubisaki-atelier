# Shizuku Engine 4.0 α — v0.2 “Nyuru”

単色の絵の具を指で「つかみ、伸ばし、離す」ための物理プロトタイプです。

## v0.2で追加した実装

- タッチ開始時、指の周囲にある絵の具粒子を最大54個だけ選択
- 選択粒子を指へ直接固定せず、柔らかいスプリングで追従
- 中央ほど強く、外側ほど弱くつかむことで自然な伸びを生成
- 伸びが限界距離を超えた粒子は指から順番に外れる
- 絵の具の総粒子数は変化せず、触るだけでは量が増えない
- Surface 1.1の平滑化・濃度面描画を維持

## テスト方法

1. `index.html` をブラウザで開く
2. 絵の具の中央付近を一度押す
3. そのまま**ゆっくり長く横へ引く**
4. 次に、同じ操作を速く行う

ゆっくり引いた場合は絵の具の一部が指について伸び、速く・遠く引くと保持が順番に外れることを確認してください。
