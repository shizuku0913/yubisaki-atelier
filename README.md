# ゆびさきアトリエ v3.0.0-M1

## Yubisaki Engine “Shizuku” — M1「ぷに」

M1では機能を増やさず、絵の具を押したときの柔らかさだけを作り直しました。

### 変更点

- 指の中心が連続的に沈む円形圧力場
- 押し出された絵の具が周囲で丸く盛り上がる
- バネ＋減衰計算による短い「ぷるん」
- セルの円を並べる描画を廃止し、補間された連続面として描画
- 凹みと盛り上がりに合わせて光と影が動く
- 空の場所では変形しない
- パレット切替、キッズ混色、既存機能を維持

### テスト

1. 絵の具を少し多めに出す
2. 絵の具の中央を1秒ほど長押しする
3. 中央が沈み、周囲が丸く盛り上がることを確認する
4. 指を離し、一度だけ柔らかく戻ることを確認する
