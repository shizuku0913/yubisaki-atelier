# ゆびさきアトリエ Ver.3.0.0 M1.6

## 修正内容
- キッズモードをUI・処理ともに削除し、リアル混色のみに統一
- 絵の具の輪郭を物理場の連続濃度面から描画し、旧ポリゴンスタンプを廃止
- 高DPI端末向けにOrganic Rendererの描画解像度を引き上げ
- 混ぜる操作でキャンバス画像をコピーしない方式へ変更
- 水を混ぜた後に触るたび色が蓄積する問題を修正
- 水の演出を非破壊化し、色レイヤーへ描き足さないよう変更

## 確認方法
1. 色を2色出す
2. 水を加える
3. 同じ場所を何度も押す／往復して混ぜる
4. 指を動かした回数だけ色が濃く増えないことを確認
