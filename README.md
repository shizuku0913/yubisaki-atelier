# ゆびさきアトリエ v3.0.0-M1.5

## Yubisaki Engine “Shizuku” — Organic Renderer

M1の「ぷに」物理を残したまま、見た目の格子感を減らす描画層へ更新しました。

### 今回の変更

- 物理グリッドを72×72へ高精細化
- 絵の具量・高さ・変形をガウシアン平滑化
- バイキュービック補間で連続面として再構成
- 輪郭を暗黙曲面として描画し、セル境界を見せない
- ごく弱い流体ワープで軸方向の角張りを抑制
- 水で薄まった絵の具を触るたびに色・濃度が加算される不具合を修正

### 確認方法

1. 絵の具を2〜3色出す
2. 水を加えて混ぜる
3. 輪郭、とくに斜め方向が以前より滑らかなことを確認する
4. 同じ場所を何度か押して離し、色や濃度が増えないことを確認する
