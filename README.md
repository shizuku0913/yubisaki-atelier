# ゆびさきアトリエ v1.6.0

子どもが指で絵の具を出し、混ぜ、作った色で絵を描けるWebアプリです。

## v1.6.0 の変更

- パレットの水分量と粘度を、お絵かき画面の筆触へ引き継ぐ
- 濃い絵の具は、盛り上がり・光沢・筆筋が残る「厚塗り」表現
- 水の多い絵の具は、透明度が上がり紙へ広がる「にじみ」表現
- 太い筆では複数の毛筋が見えるテクスチャを追加
- 線を点の連続として補間し、速く動かしても切れにくく改善
- ラメは新しい筆触にもそのまま反映
- Service Worker のキャッシュを v1.6.0 に更新

## GitHubへ反映する方法

1. ZIPを展開
2. 中身をローカルの `yubisaki-atelier` フォルダへ上書きコピー
3. GitHub Desktopで変更を確認
4. Summaryに `feat: add wet and thick finger-paint brush` と入力
5. `Commit to main` → `Push origin`

## 動作確認

- 水を入れずに混ぜた色で描くと、濃く盛り上がった筆筋になる
- 同じ色に水を多めに加えてから描くと、薄く広がる水彩風になる
- 太い筆では明るい毛筋とツヤが見える
- ラメ入りの色では筆跡にもラメが付く
