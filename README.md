# Shizuku Engine 4.0 α — v0.3 “Performance”

**Shizuku Engine** は、子どもが「本物の絵の具を指で触っている感覚」をデジタル上で再現することを目指す、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## Vision

目指しているのは、ただ線を描くアプリではありません。絵の具を指でつかみ、伸ばし、ちぎり、盛り上げる感触を、子どもが直感的に楽しめる体験です。

## v0.3 Performance Stabilization

- 粒子数を最大720個に固定し、無制限な増加を停止
- 粒子用の放射グラデーションを毎フレーム生成せず、事前生成したスプライトを再利用
- 近傍探索用の配列とSpatial Hashのバケットを再利用し、一時オブジェクトを削減
- 表面描画を最大30fpsに制限し、物理計算との負荷を分離
- 操作後に絵の具が落ち着いたら、物理計算と再描画を自動休止
- 現在のNyuruのつかむ・伸ばす挙動を維持

## テスト方法

1. `index.html` をブラウザで開く
2. 「えのぐをたす」を何度も押す
3. 上限に達するとボタンが「えのぐは いっぱい」になることを確認
4. 絵の具を繰り返しつかんで引っ張り、停止や著しい処理落ちが起きないか確認
5. 操作をやめて約1.5秒後、端末の負荷が下がることを確認
