# Shizuku Engine 4.0 α — v0.8 “Paint Budget”

**Shizuku Engine** は、子どもが「本物の絵の具を指で触っている感覚」をデジタル上で再現することを目指す、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## v0.8 Paint Budget System

子どもに「もう出せません」と言わず、内部の計算量だけを一定に保つ仕組みです。

- 1回の追加量を180粒から54粒へ変更し、少量から大きな塊まで回数で調整可能
- 物理計算の上限720粒は維持
- 新色用に90粒分の予約領域を確保
- 新しい色を選ぶと、古く静止した密集部分を優先して内部統合
- 上限付近でも追加ボタンを無効化せず、統合してから新しい絵の具を追加
- マーブルの境界・動いている筋・新しい色は統合対象から保護
- Grip、Viscosity、Paint Clump、Marble Mixing、Performance Stabilizationを維持

## テスト方法

1. 同じ色で「えのぐをたす」を数回押し、少しずつ量が増えることを確認
2. 別の色へ切り替えて何度も追加する
3. 6色を順番に何周か追加し、ボタンが「いっぱい」にならないことを確認
4. 色の境界をこね、マーブル混色が残ることを確認
5. 長時間追加・混色しても著しく重くならないことを確認

## 合否の基準

- 1回の追加量が以前より小さい
- 同じ色を押す回数で量を調整できる
- 何色追加しても追加ボタンが使える
- 粒子数は720を超えない
- 新しい色やマーブル模様が突然まとめて消えない
- 長時間遊んでも停止しない
