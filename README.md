# ゆびさきアトリエ v2.0.0

Paint Physics Foundation版。

## 追加内容
- パレットごとに64相当の格子状Paint Physics Fieldを保持
- 各セルが色・量・水分・高さ・速度を持つ
- 絵の具を出すと物理フィールドへ堆積
- 押すと中心がへこみ、周囲へ速度が伝わる
- 混ぜる方向へ絵の具の量と高さが移流
- 水を入れると濡れ具合が上がり、拡散しやすくなる
- 既存の描画機能を残したまま物理レイヤーを追加

## テスト
1. パレットを洗う
2. 赤と黄色を近くに出す
3. 境界を長押ししてからゆっくり円を描く
4. 水を加え、広がり方の変化を確認
