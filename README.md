# Shizuku Engine 4.0 α — v0.7 “Marble Mixing”

**Shizuku Engine** は、子どもが「本物の絵の具を指で触っている感覚」をデジタル上で再現することを目指す、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## v0.7 Marble Mixing 1.0

- 粒子ごとの色を実際の表面描画へ反映
- 違う色が触れただけではすぐ均一にならず、境界に色の筋が残る
- 指でこねた場所では、接触・相対速度・指の速度に応じて色素交換が速くなる
- 動かしていない場所でも、ごくゆっくりだけ色がなじむ
- 赤と青など、異なる塊を何度も折り返すことでマーブル模様から中間色へ移る
- Paint Clump、Grip、Viscosity、Performance Stabilizationを維持

## テスト方法

1. `index.html` を開く
2. 最初の赤い塊を少し横へ寄せる
3. 青を選び「えのぐをたす」を押す
4. 赤と青の境界をゆっくり往復して、赤・青・紫の筋が残るか確認
5. 円を描くように何度もこね、紫の領域が徐々に増えるか確認
6. 触れさせただけで全体が一瞬で紫にならないことを確認

## 合否の基準

- 複数色が同時に見える
- 接触直後は元の色が残る
- こねた部分から中間色が増える
- マーブル模様を経て、長くこねるほど均一色へ近づく
- 長時間操作しても著しい処理落ちや停止がない
