# Shizuku Engine 4.0 α — v0.9 “Living Paint”

**Shizuku Engine** は、子どもが「本物の絵の具を指で触っている感覚」をデジタル上で再現することを目指す、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## v0.9 Living Paint

絵の具が常に同じ性質ではなく、触り方と時間によって少しずつ変化します。

- **Softness**：こねた場所ほど柔らかくなり、長く伸びてまとわりつく
- **Dryness**：放置した場所はゆっくり硬くなり、混ざりにくく、少し切れやすくなる
- **Paint Memory**：触れた場所は動きと形の記憶を短時間保持し、指を離したあともゆっくり落ち着く
- **Thickness**：粒子を増やさず、密集度から厚みを推定して盛り上がりを表現
- 柔らかさ・乾き・厚みを、Grip、Clump結合、混色速度、切れ方、描画へ反映
- 完全乾燥はせず、子どもがいつでも再びこねて遊べる範囲に制限
- v0.8 Paint Budget Systemと720粒の上限を維持

## テスト方法

1. 新しい絵の具をゆっくり何度もこね、伸び方が徐々に柔らかくなるか確認
2. 一部を数秒放置し、触り続けた部分と引っ張り方・混ざり方を比べる
3. 絵の具を同じ場所へ重ね、厚い部分が少し大きく重く見えるか確認
4. 指を離したあと、絵の具が急停止せず短く余韻を残して落ち着くか確認
5. 複数色を混ぜ、放置部分よりこねた部分の方が混ざりやすいか確認

## 合否の基準

- こね続けた場所が伸びやすく感じられる
- 放置した場所がわずかに締まり、混ざりにくく感じられる
- 厚い部分が平坦な部分より盛り上がって見える
- 性質変化が強すぎず、子どもの操作を妨げない
- Paint Budget、Grip、Viscosity、Clump、Marble Mixingが壊れていない
