# Shizuku Engine 4.0 α — v0.9.1 “Surface Deformation”

**Shizuku Engine** は、子どもが本物の絵の具を指で触る感覚をデジタル上で再現する、**ゆびさきあとりえ**専用のペイント物理エンジンです。

## v0.9.1 Surface Deformation

Living Paintの状態値を、表面の形と光へ明確に反映しました。

- 密集した部分ほど厚く盛り上がる
- 指で押した中心は低くなり、その周囲へ絵の具が逃げて縁が盛り上がる
- 引っ張られたClump結合を連続面として描き、塊と指の間に細い「首」を形成
- 伸びと速度に応じて粒子の描画をわずかに長くし、流れの方向を見せる
- 厚みからハイライトと反対側の影を生成し、平面ではなく変形する表面として表示
- Paint Budget、Grip、Viscosity、Clump、Marble Mixing、Living Paintを維持

## テスト方法

1. 大きな塊の中央をゆっくり押し、中央が沈んで周囲が盛り上がって見えるか確認
2. 塊の端をゆっくり引き、元の塊との間に細い首が形成されるか確認
3. 同じ場所へ絵の具を何度か追加し、厚い部分の光と影が強くなるか確認
4. 赤と青を混ぜ、首の部分にも混色が連続して表示されるか確認
5. 色を何度も追加しても、v0.8と同様に止まらないか確認

## 合否の基準

- 押した場所と盛り上がった場所の差が目で分かる
- ゆっくり引いたとき、点の列ではなく細い絵の具の首に見える
- 厚み表現が混色を隠さない
- 長時間操作で著しく重くならない
