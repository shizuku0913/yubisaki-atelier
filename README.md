# Shizuku Rebuild M1

描画・触感エンジンを旧コードから切り離して新規実装した、単色絵の具の検証版です。

## M1で確認すること

- 輪郭が多角形ではなく連続した密度場から生成される
- 指で押すと中心が沈み、周囲へ既存の絵の具が逃げる
- 指を離すと短く戻る
- タッチ操作では絵の具の密度を追加しない
- 色や絵の具量を増やすのは「えのぐを だす」ボタンだけ

## 旧エンジンから削除したもの

- `blobPath()` などの多角形輪郭
- 周辺画像を `drawImage()` でコピーする変形
- キッズ／リアル混色の分岐
- 色キャンバスと物理キャンバスの二重管理

## 起動

`index.html` を静的WebサーバーまたはGitHub Pagesで開いてください。
