# ゆびさきアトリエ v3.0.0-alpha1

## Yubisaki Engine “Shizuku”

Ver.3 系の最初の実装です。今回のテーマは **Soft Body**。

### 変更点

- 指の中心が丸く沈み、周囲が柔らかく盛り上がる圧力モデルへ変更
- 指を離したあとに小さく揺れて戻る「ぷるん」の減衰振動を追加
- 格子単位の移動をやめ、補間を使った滑らかな流れへ変更
- 低速では盛り上がり、高速では薄く伸びる反応を追加
- 物理レイヤーを点の集合ではなく、連続した柔らかな面として描画
- 既存のキッズ混色モードとパレット切替修正を維持

### 確認方法

1. 絵の具をパレットへ出す
2. 同じ場所を短く押して離し、「ぷに → ぷるん」を確認
3. ゆっくり引っ張った場合と、素早く払った場合を比較
4. 円を描き、角張らずに曲がるか確認

これは alpha1 です。次は Finger Dynamics と渦の流れを調整します。
