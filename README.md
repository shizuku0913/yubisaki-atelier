# ゆびさきアトリエ v2.1.1

## キッズ混色モード

- 初期設定は「キッズ」
- 混ぜ始めは少しマーブルが残る
- 途中から計算済みの完成色へ滑らかにまとまる
- 従来より早く、はっきりした1色として完成する
- 設定から「リアル」に切り替えると、マーブルが長く残る
- 完成時は物理レイヤーも同じ色へ統一されるため、色の粒が残らない

## テスト
赤と黄色を近くに出し、境界を円を描くように混ぜてください。キッズモードでは途中に短いマーブルを経て、明確なオレンジへまとまります。
